# Ejercicio JSON

## Problema escogido

El problema escogido para realizar el ejercicio fue representar información básica mediante archivos JSON.

Se trabajaron dos estructuras principales:

1. **Usuarios:** se creó un archivo JSON con información de usuarios, incluyendo identificador, nombre y edad.
2. **Producto:** se creó un archivo JSON con la información de un producto, incluyendo identificador, nombre, categoría, precio, stock y disponibilidad.

El objetivo del ejercicio fue practicar la creación y organización de datos en formato JSON, utilizando estructuras como objetos, arreglos, cadenas de texto, números y valores booleanos.

## Archivos realizados durante la formación

* `ejercicio1.json`: contiene información de usuarios.
* `ejercicio2.json`: contiene información de un producto.
